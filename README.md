# Birthday Greeting Campaign - Salesforce Marketing Cloud

This project demonstrates a personalized birthday email campaign in SFMC using AMPscript, Data Extensions, and Automation Studio.

## Features
- Sends birthday greetings to subscribers.
- Personalizes content using AMPscript.
- Uses Data Extension for birthday tracking.
- Ready for Automation Studio implementation.

## Contents
- Email template with AMPscript.
- Sample Data Extension (CSV format).
- AMPscript logic.
- Automation Studio setup steps.

## Technologies
- Salesforce Marketing Cloud.
- AMPscript.
- HTML.
- Email Studio & Automation Studio.

## How to Use
1. Import **birthday_subscribers.csv** as a DE.
2. Create a new email using **birthday_email.html**.
3. Insert the AMPscript block from **ampscript_snippet.ampscript**.
4. Set up automation to run daily and filter subscribers whose birthday is today.
